# Microsoft Build //localhost - Wellington

**Date:** 18 June 2026

Join us in Wellington for a focused, in-person session bringing the key insights from Microsoft Build 2026 to the Wellington AI community.

Spaces are limited for this in-person event. Please register: [https://developer.microsoft.com/en-us/reactor/events/27230/](https://developer.microsoft.com/en-us/reactor/events/27230/)

This event is designed for developers, cloud engineers, AI practitioners, and technical leaders across government agencies and startup teams, innovators, and the wider developer community who are looking to move from experimentation to real-world implementation of AI.

You'll explore how to design, build, and deploy AI-powered solutions on Azure using tools like Microsoft Foundry and GitHub Copilot, with an emphasis on practical application and secure, scalable approaches relevant to the NZ context.

## What You Can Expect

- Key takeaways and announcements from Microsoft Build 2026
- Real-world AI and generative AI use cases
- Live demos and guided, hands-on labs
- Best practices for building and deploying AI applications
- Opportunities to connect with peers across the Wellington developer community

- Exploring AI use cases that's hands on from ideation to deployment
