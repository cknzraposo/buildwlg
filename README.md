# Microsoft Build //localhost - Wellington

Microsoft Build //localhost - Wellington is a focused in-person event that brings key Microsoft Build 2026 announcements to the Wellington AI and developer community.

## Event Purpose

This event is designed to help technical teams move from AI experimentation to real-world implementation. Sessions and labs focus on practical delivery with Azure, Microsoft Foundry, and GitHub Copilot.

## Who Should Attend

- Developers building new digital services
- Cloud engineers modernising platforms on Azure
- AI practitioners moving from prototype to production
- Technical leaders shaping architecture and governance

## What To Expect

- Curated Microsoft Build 2026 takeaways
- Real-world AI and generative AI use cases
- Guided hands-on labs and live demos
- Practical patterns for secure, scalable deployment
- Opportunities to connect with the Wellington developer community

## Schedule Highlights

The program includes breakout sessions and hands-on labs across:

- Context-aware agents and enterprise intelligence
- Deploying AI solutions on Azure from zero to production
- Observability, evaluation, and hosted-agent quality
- Model selection and implementation using Microsoft Foundry

## Speakers

Xiaodi Yan, Chandima Kulathilake, Zack Yang, Jerome Brown, Shobhit Bhatnagar
