Access

How to access LAB 501

  1. Open https://buildlocalhost.learnondemand.net/ in your browser.
  2. Sign in, or create an account if you do not already have one.
  3. Enter the lab access key exactly as shown:
    5194A9CE6942A496
  4. Select LAB 501 from your available labs.
  5. Select Start and wait for the lab environment to finish provisioning.
  6. Open the LAB 501 GitHub repo and follow the README steps in order.

Tip: Keep both the lab platform and the GitHub repository open in separate tabs.

Troubleshooting

If something does not work

  • Re-enter the key and check for accidental spaces when pasting.
  • Confirm you are signed in to the lab platform account you registered with.
  • Refresh the lab page if provisioning appears stuck for an extended period.
  • Ask an event host for support if access or provisioning still fails.
